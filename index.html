입력 방식 안내
  • 키보드: 방향키 또는 WASD
  • 모바일: 화면 D-pad 터치
  • 음성: “앞으로”, “뒤로”, “왼쪽”, “오른쪽”, “먹기”
  • DotPad 패닝키: 방향 입력과 동일한 movePlayer(direction) 구조에 연결 가능

Prototype note: sendDotPadFrame(matrix) 함수가 실제 DotPadSDK-1.0.0.js 연결 지점입니다.
